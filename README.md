# Task_Manager
Task Manager App
---------------------------------
A simple and user-friendly Task Manager web application that allows users to add, update, and delete tasks easily.
This project was built to practice frontend development concepts and improve understanding of state management and user interactions.

Features
------------------------------
Add new tasks
Update existing tasks
Delete tasks
View Tasks

Technologies Used
-----------------------------
React Js + Vite
TypeScript

How It Works
------------------------
_> User enters a task in the input field.
-> Clicks Add to store the task.
-> Task appears in the list.
-> User can: Add the Task, Edit the task (update functionality), Delete the task

The UI updates dynamically without refreshing the page.
